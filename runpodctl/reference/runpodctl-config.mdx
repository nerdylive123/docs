## runpodctl config

CLI Config

### Synopsis

RunPod CLI Config Settings

```sh
runpodctl config [flags]
```

### Options

```text
    --apiKey string   RunPod API key
    --apiUrl string   RunPod API URL (default "https://api.runpod.io/graphql")
-h, --help            help for config
```
